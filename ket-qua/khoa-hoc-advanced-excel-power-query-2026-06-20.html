From Broken Data to Numbers You Can Trust
Từ dữ liệu lỗi đến con số đáng tin cậy
08:30 – 16:30
☀ Buổi sáng · 08:30 – 11:30
When formulas look right but numbers are wrong — where does the error start?Công thức kéo đúng mà vẫn ra số sai — lỗi bắt đầu từ đâu?
Session 1: Data Foundations & Cleaning
Buổi 1: Nền tảng dữ liệu & làm sạch
1 Understand the Data Model Hiểu mô hình dữ liệu — nền tảng mọi báo cáo
  • Grain: what does one row represent? — every report must answer this firstGrain: 1 dòng nói lên điều gì? — mọi báo cáo phải trả lời câu này trước tiên
  • Dimension table: lookup / classification data — Product, Customer, RegionBảng Dimension: dữ liệu phân loại — Sản phẩm, Khách hàng, Khu vực
  • Fact table: transaction / measurement data — the source that feeds every formulaBảng Fact: dữ liệu giao dịch, số đo lường — nguồn cung cấp cho mọi công thức
2 Detect & Repair Structural Errors Phát hiện & sửa lỗi cấu trúc
  • Merged cells, multi-row headers, blank rows inside data — the silent formula-killersÔ gộp, tiêu đề nhiều dòng, dòng trống chen giữa bảng — nguyên nhân 90% công thức báo sai
  • Ctrl+G → Go To Special → Blanks: select ALL blanks at once, fill or delete in one moveCtrl+G → Go To Special → Blanks: chọn toàn bộ ô trống cùng lúc, điền/xóa 1 thao tác
  • Numbers stored as text: identify, convert — SUMIFS and Pivot count them correctlySố lưu dạng văn bản: nhận biết và chuyển đổi — SUMIFS, Pivot tính đúng ngay
3 Enforce Team-Wide Data Standards Thiết lập chuẩn nhập liệu toàn đội
  • Format Cells: number, date, currency, custom codes — one visual standard for allFormat Cells: số, ngày, tiền tệ, mã tùy chỉnh — toàn đội dùng chung 1 chuẩn hiển thị
  • Data Validation: dropdown list, date range, custom formula — block wrong input at sourceData Validation: danh sách xổ, kiểm tra ngày, công thức — chặn nhập sai từ nguồn, không sửa sau
Applied Tools
Ctrl+G Go To SpecialFormat Cells (Custom, Date, Currency)Data ValidationText-to-ColumnsFlash FillRemove Duplicates

Session 2: Lookup & Data Enrichment
Buổi 2: Tra cứu & Mở rộng dữ liệu
1 Replace VLOOKUP With XLOOKUP — Permanently Thay VLOOKUP bằng XLOOKUP — vĩnh viễn
  • XLOOKUP: look up by any key, return any column, any direction — no column counting ever againXLOOKUP: tra theo bất kỳ khoá, trả về bất kỳ cột, bất kỳ hướng — không đếm cột, không lỗi khi chèn cột
  • Enrich Fact table from Dim tables: pull Name, Phone, Category, Price automaticallyMở rộng bảng Fact từ bảng Dim: lấy Tên, SĐT, Danh mục, Giá tự động — mô hình Fact–Dim thực tế
2 Multi-Condition Aggregation: SUMIFS & COUNTIFS Tổng hợp đa điều kiện
  • SUMIFS: aggregate by department × month × product simultaneously in one formulaSUMIFS: tổng hợp theo bộ phận × tháng × sản phẩm cùng lúc trong 1 công thức duy nhất
  • IF / IFS: flag missed targets, classify performance tiers — decision logic inside the cellIF / IFS: đánh dấu không đạt, phân hạng hiệu suất — logic quyết định nằm ngay trong ô
3 Dynamic Arrays + Preview of the Next Level Mảng động + nhìn trước cấp độ tiếp theo
  • FILTER + UNIQUE: lists that update themselves automatically when data changesFILTER + UNIQUE: danh sách tự cập nhật khi dữ liệu nguồn thay đổi — không lọc thủ công
  • Preview → Merge Queries (Power Query, Day 3): across multiple files, this replaces XLOOKUP at any scaleNhìn trước → Merge Queries: khi dữ liệu trải nhiều file, thay XLOOKUP ở quy mô lớn
Applied Tools
XLOOKUPSUMIFSCOUNTIFSIF / IFSFILTERUNIQUEIFERROR

Session 3: Pivot Table Mastery
Buổi 3: Thành thạo Pivot Table — mọi báo cáo trong vài phút
1 Build a Report-Ready Pivot in Minutes Xây Pivot chuẩn báo cáo trong vài phút
  • Tabular layout, number format, hide Field Headers — output you can paste into any slide immediatelyTabular layout, định dạng số, ẩn Field Headers — kết quả paste vào slide ngay được
  • Top 10 filter: best customers by revenue + order count, with name and region auto-populatedLọc Top 10: khách đầu bảng theo doanh thu + số đơn, tên và tỉnh/thành tự điền
2 Advanced Analysis Built Into Pivot Phân tích nâng cao ngay trong Pivot
  • Show Value As: % of Grand Total, % of Row, % of Column — contribution share without extra formulasShow Value As: % tổng, % dòng, % cột — tỷ trọng hiển thị ngay, không cần công thức bổ sung
  • Difference From & % Difference From: MoM and YoY growth built directly into PivotDifference From & % Difference From: tăng trưởng MoM, YoY có sẵn trong Pivot — không cần cột phụ
  • Group by date, Running Total — time-series analysis with a few clicksGroup theo ngày, Running Total — phân tích chuỗi thời gian chỉ với vài cú click
3 Make the Report Fully Interactive Biến báo cáo thành báo cáo tương tác
  • Slicer: one visual filter — click Region, all pivot views update instantlySlicer: 1 bộ lọc trực quan — click chọn Vùng, toàn bộ Pivot cập nhật tức thì
  • Report Connections: one Slicer controls multiple Pivot Tables simultaneouslyReport Connections: 1 Slicer điều khiển nhiều Pivot cùng lúc — mọi bảng đồng bộ

Session 4: Charts & Dashboard
Buổi 4: Biểu đồ & Dashboard tương tác
1 Choose & Format the Right Chart Chọn đúng biểu đồ & format chuyên nghiệp
  • Column (compare), Bar (rank), Line (trend), Combo dual-axis (KPI + growth on one chart)Column (so sánh), Bar (xếp hạng), Line (xu hướng), Combo 2 trục (KPI + tăng trưởng trên 1 biểu đồ)
  • Format mastery: consistent palette, axis scale, gap width, data labels — production-quality in 5 minutesFormat nâng cao: bộ màu nhất quán, thang trục, gap width, nhãn số liệu — biểu đồ chuyên nghiệp trong 5 phút
2 Creative Charts That Stand Out Biểu đồ sáng tạo — khác biệt so với báo cáo thông thường
  • Doughnut progress ring, stacked % bar, target-line column, Waterfall — advanced visual storytellingVòng tiến độ Doughnut, thanh tỷ trọng, cột kèm kế hoạch, Waterfall — kể chuyện bằng hình ảnh nâng cao
  • Build 6 chart types from one dataset — creative production-quality charts in one sessionXây 6 kiểu biểu đồ từ cùng 1 bộ dữ liệu — chất lượng thực chiến, hoàn thành trong 1 buổi
3 Design & Build the Full Interactive Dashboard Thiết kế & xây Dashboard tương tác hoàn chỉnh
  • Layout design first: header zone · KPI card zone · chart area · slicer panel — plan before buildThiết kế layout trước: vùng header, thẻ KPI, biểu đồ, slicer — phác thảo trước khi xây
  • KPI cards: #,##0.0,,"M" number format · ▲▼ icon conditional formatting · vs-target badgeThẻ KPI chuẩn: định dạng M, icon ▲▼ màu điều kiện, badge so sánh kế hoạch
  • One Slicer filters all KPIs + all charts simultaneously — connected via Report Connections1 Slicer lọc tất cả KPI + tất cả biểu đồ cùng lúc — kết nối qua Report Connections

10 branch files every month, all different formats — consolidate by hand, then start over next month?10 file chi nhánh mỗi tháng, định dạng khác nhau — gộp tay xong, tháng sau lại làm lại từ đầu?
Session 5: Power Query — Tame the Uninvited Guest
Buổi 5: Power Query — Thuần hoá "khách không mời mà tới"
1 Power Query Overview + From Folder Tổng quan Power Query + From Folder — nền tảng tự động hoá
  • Power Query is the automation layer between raw data and your report — every step recorded, re-run on RefreshPower Query là lớp tự động giữa dữ liệu thô và báo cáo — mọi bước ghi lại, nhấn Refresh là chạy lại
  • "Uninvited guest" case: From Folder auto-detects all xlsx — add a new branch file, Refresh picks it up instantlyBài toán "khách không mời mà tới": From Folder tự nhận diện mọi file xlsx — thêm file, Refresh là có ngay
  • Append Queries: stack 12 monthly files into one clean consolidated table — 3 seconds, not 3 hoursAppend Queries: chồng 12 file tháng thành 1 bảng tổng hợp — 3 giây thay 3 giờ làm tay
2 Transform Pipeline — Define Once, Run Every Month Pipeline xử lý — định nghĩa 1 lần, chạy mãi mãi
  • Filter rows, Remove columns, Rename headers, Change Type — chain steps into a reusable pipelineFilter, Remove Columns, Rename, Change Type — chuỗi các bước thành pipeline tái sử dụng mỗi tháng
  • Unpivot Columns: reshape wide pivot-style files (months as columns) into flat, workable dataUnpivot Columns: chuyển file rộng (tháng trải thành cột) thành bảng phẳng — kỹ thuật "cứu" file sai cấu trúc
3 Merge Queries & M Code — Handle Any File Structure Merge Queries & M Code — xử lý mọi cấu trúc file
  • Merge Queries: join Fact table with Dim tables by key — XLOOKUP at any scale, fully automatic on RefreshMerge Queries: ghép Fact với Dim theo khoá — như XLOOKUP nhưng triệu dòng, nhiều file, tự refresh
  • M code for flexibility: Table.Combine, Excel.Workbook, Table.SelectRows, Text.ContainsM code linh hoạt: làm những gì giao diện không làm được

Session 6: Power BI & DAX + Claude AI / Copilot
Buổi 6: Power BI & DAX + Claude AI / Copilot — Hệ thống báo cáo hoàn chỉnh
1 Power BI Data Model — Replace All VLOOKUP Mô hình dữ liệu Power BI — thay toàn bộ VLOOKUP bằng 1 kết nối
  • Power BI Desktop: Report · Data · Model — understand the 3-layer workspace in 15 minutesPower BI Desktop: 3 giao diện Report / Data / Model — nắm vững kiến trúc trong 15 phút đầu
  • Define Fact–Dim relationships: one persistent link that powers all measures across the reportQuan hệ Fact–Dim: 1 kết nối duy trì thay toàn bộ VLOOKUP — mọi measure tự tính đúng theo chiều lọc
  • Build interactive visuals: KPI cards, bar/line charts, slicers — cross-filter by designXây visuals tương tác: thẻ KPI, cột/đường, slicer — tự lọc chéo nhau theo mặc định
2 DAX Measures — Compute What Pivot Cannot DAX Measures — tính những gì Pivot không bao giờ làm được
  • CALCULATE: override filter context — compute revenue for a specific segment regardless of slicerCALCULATE: ghi đè filter context — tính doanh thu cho 1 phân khúc cụ thể bất kể slicer đang lọc gì
  • SUMX: row-by-row calculation — margin per transaction, weighted averages at any granularitySUMX: tính từng dòng — margin mỗi giao dịch, trung bình có trọng số ở bất kỳ độ chi tiết nào
  • DIVIDE (safe division) · DATEADD (YoY, MoM) — KPI measures that auto-recalculate on every slicer clickDIVIDE (chia an toàn) · DATEADD (YoY, MoM) — measures KPI tự tính lại theo từng click slicer
3 AI-Powered Productivity: Claude AI + Copilot Tăng năng suất bằng AI: Claude AI + Copilot trong Excel
  • Describe in plain language → get any SUMIFS, XLOOKUP, or M code in seconds — no syntax memorisationMô tả bằng tiếng Việt → ra SUMIFS, XLOOKUP, M code trong vài giây — không cần nhớ cú pháp
  • Generate DAX from a business question: 'calculate YoY growth by region' → CALCULATE + DATEADDSoạn DAX từ câu hỏi nghiệp vụ: 'tính tăng trưởng YoY theo vùng' → CALCULATE + DATEADD ngay
  • Explain any formula, debug M code, review data model logic — AI as your co-analyst, available 24/7Giải thích công thức khó, debug M code, kiểm tra mô hình dữ liệu — AI như người cộng sự phân tích 24/7
